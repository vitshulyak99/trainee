Задание 4

В отдельной папке (./File относительно корня приложения) есть файл с текстом. Засуньте туда любой, не очень большой текст.

Пользователь вводит с клавиатуры любое количество символов, разделённых запятой. Потом вводит целое число N.

Для каждого введённого символа вывести на экран данные типа "File has [number] [symbol] symbols at [FileLastModifiedDate]"

Каждые N считываний программа должна добавлять в этот файл один случайный символ из тех, что ввёл пользователь.

Чтение количества символов и запись реализовать асинхронными вычислениями. Можете потоки создавать, можете TPL юзать.

В отдельном (!) проекте написать Unit тесты с использованием xUnit и moq. По 3-4 теста на два-три разных класса. UI не тестировать.
